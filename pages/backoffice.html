BackOffice

Construido en el lenguaje Java sobre los productos open source lideres de la industria bajo licencia Apache Software v2.0 o MIT. Framework de prototipado rápido basado en el enfoque DDD con el que se puede hacer aplicaciones web de forma fácil, sencilla y escalable.
Framework open source para aplicaciones web orientado a componentes escrito en Java utilizado por Apache ISIS para crear la interface de usuario en base a las entidades del Dominio.
Es una tecnología para crear aplicaciones autocontenidas. De este modo, el equipo se puede centrar en el desarrollo a medida. Dejando a un lado la arquitectura. Es decir, Apache ISIS delega en Spring Boot las labores de configuración de dependencias, Rest API y el despliegue del servicio de aplicaciones.
